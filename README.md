## 📌 Sobre o projeto

O **CORE v1.0** é um projeto experimental focado na construção de uma base arquitetural sólida para sistemas inteligentes e escaláveis.

O objetivo é desenvolver um sistema com foco em:

* Arquitetura limpa (Clean Architecture)
* Separação de responsabilidades
* Escalabilidade
* Integração futura com Inteligência Artificial
* Estrutura preparada para evolução contínua

---

## 🏗 Arquitetura

O projeto segue princípios inspirados em **Clean Architecture**, organizando as responsabilidades em diferentes camadas.

```
CORE.Api
CORE.Application
CORE.Domain
CORE.Infrastructure
```

### Camadas

**CORE.Domain**

* Entidades
* Regras de negócio
* Modelos do domínio

**CORE.Application**

* Casos de uso
* Interfaces
* Regras de aplicação

**CORE.Infrastructure**

* Implementações técnicas
* Banco de dados
* Integrações externas

**CORE.Api**

* Camada de entrada do sistema
* Controllers
* Requests e Responses
* Configuração da aplicação
